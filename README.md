# Personal Finance Tracker

A simple command-line application to track your income and expenses.

## Features
- Add income and expense transactions.
- View transaction history.
- View financial summary (Total Income, Total Expense, Net Balance).
- Data is automatically saved to `data.json`.

## How to Run
1. Make sure you have Python installed.
2. Run the script:
   ```bash
   python finance_tracker.py
   ```
3. Follow the on-screen prompts.

## Files
- `finance_tracker.py`: Main application script.
- `data.json`: Stores your transaction data (created automatically).
